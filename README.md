## Cara Menjalankan
 1.Download file zip
 
 2.Buka menggunakan Vscode
 
 3.Buka terminal Vscode

 4.Ketikkan "npm install" untuk menginstall folder node modules
 
 5.Jalankan dengan perintah "npm run start-dev"
